To get started, take a look at src/app/page.tsx.

## Website ko Live Kaise Karein (Deployment Guide)

Aap in simple steps ko follow karke apni website ko live kar sakte hain. Aapko iske liye billing account ki zaroorat nahi padegi. Hum Vercel ya Netlify jaise free platforms ka istemal karenge.

### Step 1: Code ko GitHub par Daalein

Sabse pehle, aapko is project ke saare code ko apne GitHub account par ek "repository" mein daalna hoga. Agar aapne yeh kar liya hai, to agle step par jaayein.

1.  **GitHub par Repository Banayein:** Apne GitHub account mein login karein aur ek nayi (new) repository banayein. Usse koi bhi naam de sakte hain (jaise "my-first-app").
2.  **Code Upload Karein:** Is platform se code ko download karke (ya copy-paste karke) apne computer par layein aur phir use us nayi GitHub repository mein upload ("push") kar dein.

### Step 2: Vercel (Recommended) ya Netlify par Deploy Karein

**Vercel (Aasan aur Next.js ke liye Best):**

1.  **Vercel par Sign Up Karein:** [https://vercel.com](https://vercel.com) par jaayein aur apne GitHub account se "Continue with GitHub" par click karke sign up/login karein.
2.  **Project Import Karein:** Login hone ke baad, aapko "Import Git Repository" ka option dikhega. Yahan apni GitHub repository ko select karein jisme aapne code daala hai.
3.  **Deploy par Click Karein:** Vercel apne aap sab settings pehchan lega. Aapko bas "Deploy" button par click karna hai.

Kuch hi der mein, Vercel aapki website ko live kar dega aur aapko ek link de dega!

**Netlify (Ek aur accha vikalp):**

1.  **Netlify par Sign Up Karein:** [https://www.netlify.com](https://www.netlify.com) par jaayein aur apne GitHub account se sign up/login karein.
2.  **Project Import Karein:** Dashboard par, "Add new site" -> "Import an existing project" par click karein.
3.  **Repository Chunein:** GitHub se apni code wali repository ko chunein.
4.  **Deploy Karein:** Settings ko default rehne dein aur "Deploy site" button par click karein.

Thodi der mein, Netlify bhi aapki website ka live link generate kar dega.

In dono platforms ka fayda yeh hai ki jab bhi aap apne GitHub repository mein naya code daalenge, aapki website apne aap update ho jaayegi.
