# Firebase Studio

This is a NextJS starter in Firebase Studio.

To get started, take a look at src/app/page.tsx.

## Website ko Live Kaise Karein (Deployment Guide)

Aap in simple steps ko follow karke apni website ko live kar sakte hain. Yeh project Firebase App Hosting ke liye pehle se taiyaar hai, jo deployment ko bahut aasan bana deta hai.

**Step 1: Code ko GitHub par Daalein**

Sabse pehle, aapko is project ke saare code ko apne GitHub account par ek "repository" mein daalna hoga.

1.  **GitHub par Repository Banayein:** Apne GitHub account mein login karein aur ek nayi (new) repository banayein. Usse koi bhi naam de sakte hain (jaise "my-first-app").
2.  **Code Upload Karein:** Is platform se code ko download karke (ya copy-paste karke) apne computer par layein aur phir use us nayi GitHub repository mein upload ("push") kar dein.

**Step 2: Firebase Console mein Jaayein**

Apne web browser mein Firebase Console kholein: [https://console.firebase.google.com/](https://console.firebase.google.com/)

Apne uss Firebase project ko chunein jisse aap yeh app connect karna chahte hain.

**Step 3: Firebase App Hosting ko GitHub se Connect Karein**

1.  Firebase Console ke side menu mein, **Build** section ke andar **App Hosting** par click karein.
2.  Aapko "Get Started" ya "Connect" jaisa button dikhega. Uspar click karein.
3.  Yeh aapse aapka **GitHub account connect** karne ke liye kahega. Instructions ko follow karein.
4.  GitHub connect hone ke baad, yeh aapse puchega ki **konsi repository** connect karni hai. Aapne Step 1 mein jo repository banayi thi, usse select karein.

**Step 4: Ho Gaya! Link Mil Jaayega**

Jaise hi aap repository connect karenge, Firebase App Hosting aapke code ko automatically build karega aur use live kar dega.

Process poora hone par, App Hosting ke dashboard par hi aapko aapki **live website ka URL (link)** mil jaayega.

Ab se, jab bhi aap apne GitHub repository mein koi naya code daalenge, aapki website apne aap update ho jaayegi!
